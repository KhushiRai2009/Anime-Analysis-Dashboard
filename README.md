# Anime-Analysis-Dashboard

This repository contains an in-depth analysis of anime datasets using Python and Tableau. The project focuses on extracting meaningful insights from anime-related data, including ratings, genres, episodes, and user engagement. Interactive dashboards and visualizations help identify trends, popular genres, audience preferences, and patterns in anime production over time.

Key Features:

Comprehensive data cleaning and preprocessing.

Interactive dashboards for metrics like total anime count, average ratings, and member engagement.

Advanced visualizations including KPI charts, sparkline trends, and dumbbell comparisons.

Insights into genre popularity, seasonal trends, and rating distribution.


Tools Used: Python (pandas, matplotlib, seaborn), Tableau, Excel.

Objective: To provide anime enthusiasts and analysts with actionable insights and visual storytelling through clean data visualization and exploratory analysis.
